Big Data Project

From Raw Data to Business Insight

Processing millions of data, we performed big data analytics skills to dig out the valuable information lie in the data.

Hadoop - Big Data Distributed Processing Framework

Hadoop framework is implemented to handle the huge datasets. Hive, PySpark and SparklyR are used for their high data processes efficiency.

The Big Data

The datasets contain more than 700,000,000 user behavior record on the TaoBao e-commerce. Covering more than 1M users and 800K advertisements.

Data Preprocessing

Dealing with the missing values, outlier, and data that have ‘weird’ time interval. Enrich the data with vital features from different sources.
